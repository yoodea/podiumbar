# PODIUM BAR — QR-меню

Онлайн-меню ресто-бара PODIUM. Статический сайт: HTML + JS, без сервера.

**Прод:** https://yoodea.github.io/podiumbar/

## Как редактировать меню

Все блюда, цены и переводы — в [index.html](index.html), внутри `static MENU = [...]`:

- `n` — название, `p` — цена в тенге, `note` — примечание (объём и т.п.)
- `b: 'hit'` / `b: 'new'` — бейдж «ХИТ» / «НОВИНКА»
- `v: [{ l: '50 мл', p: 990 }, ...]` — варианты объёма с разными ценами
- Названия категорий переводятся в `name: { ru, kz, en }`

Тексты интерфейса (три языка) — в `static T = {...}` там же.

После правки: `git add -A && git commit -m "menu update" && git push` — сайт обновится за ~1 минуту.

## Локальный запуск

```
python3 -m http.server 4173
# открыть http://localhost:4173
```

## QR-код

`qr-menu.png` / `qr-menu.svg` в корне — ведут на прод-адрес. Печатать можно прямо их.
